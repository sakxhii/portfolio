Projects

  • StudyNexus: Virtual Classroom – A Java Servlet-based platform for online education. It enables real-time video lectures, secure assignment submissions, and interactive student-teacher discussions.

  • HealthPulse – A Smart Health Prediction System powered by machine learning models. It analyzes user health data and predicts potential risks, offering proactive care suggestions. Built as an academic project, it emphasized the role of AI in improving public health outcomes.

  • Imagify – A SaaS text-to-image generator built with the MERN stack and integrated with the Clipdrop AI API. Features include a credit-based usage system and online payment gateway.

  • SignifyAI – A sign language detection system created with TensorFlow, OpenCV, and Flask. It recognizes hand gestures in real-time, bridging the communication gap for hearing-impaired users. This project nurtured my interest in computer vision and inclusive technologies.
